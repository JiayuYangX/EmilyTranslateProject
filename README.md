# Emily/Phase4.5 官方人格部分汉化

伪春菜人格 Ghost：Emily4.5 的中文汉化工程
原汉化作者：Dark Angel

复刻自 <https://github.com/DarkAngelZT/EmilyTranslateProject>。同步官方版上游更新；修改所有文件编码为 UTF-8，确保编码正确。

## 安装方法：

从 Release 中下载最新版本提供的 NAR 文件。

- 安装方案 1：打开 SSP，将 NAR 文件拖入，点击确定覆盖

- 安装方案 2：将 NAR 文件的默认打开方式设为 SSP，双击运行，点击确定覆盖

- 安装方案 3：将 NAR 文件的后缀名改为 .zip，解压覆盖到 SSP/ghost/emily4 目录下

## 注意事项：

记得在设置中关闭人格自动更新，否则汉化内容会被覆盖。
