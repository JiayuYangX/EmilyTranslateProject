# Emily/Phase4.5 官方人格完全汉化

SSP 伪春菜默认人格（Ghost） **Emily/Phase4.5** 的汉化工程，支持在线更新。  
**汉化作者**：Jiayu Yang  
**原始项目作者**：Dark Angel

大部分翻译内容由大模型完成，主菜单及其他少部分汉化内容取自 <https://github.com/DarkAngelZT/EmilyTranslateProject>；修改所有文件编码为 UTF-8，确保编码正确。

## 汉化范围

- **36 个词典文件**：`ghost/master/*.dic`、`ghost/master/system/*.dic` 注释与对话全部汉化
- **人格配置文件**：`ghost/master/*.txt`、`ghost/master/mop/0unit.txt`
- **外壳配置文件**：`shell/master/*.txt`
- **README 文档**：`readme.txt`

## 安装方法：

从 Release 中下载最新版本提供的 NAR 文件。

- **安装方案 1**：打开 SSP，将 NAR 文件拖入，点击确定覆盖

- **安装方案 2**：将 NAR 文件的默认打开方式设为 SSP，双击运行，点击确定覆盖

- **安装方案 3**：将 NAR 文件的后缀名改为 .zip，解压覆盖到 SSP/ghost/emily4 目录下
